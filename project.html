Reminder App

The Reminder App is a web-based application built using HTML, CSS, JavaScript, Django, MySQL, and Python. It helps users create, manage, and track their reminders efficiently. Users can set reminders for tasks, appointments, or events and receive notifications when due.

  • ✅ User Authentication – Secure login and logout functionality
  • ✅ Add, Edit, Delete Reminders – Users can manage their reminders easily
  • ✅ Database Management – Store and retrieve reminders using MySQL
  • ✅ Django Backend – Handles data processing and business logic
  • ✅ REST API Support – Can integrate with other apps (if required)

E-Commerce Website

  • ✅ User registration and secure authentication
  • ✅ Product listing with search and filter functionality

  • ✅ Admin dashboard for managing products, users, and orders
  • ✅ Reviews and ratings for products

Anime Streaming App

The Anime World project is a dynamic web application that combines several interactive features using only HTML, CSS, and JavaScript. Key features of the project include:

  • ✅ Stream App: A section where users can view anime streaming content or links to different anime shows. This could include embedded videos or external streaming links.
  • ✅ Gallery: A section displaying various anime images or posters, where users can browse through a collection of anime visuals in a grid layout.
  • ✅ Typing Game: An engaging typing game that challenges users to type anime-related words or phrases quickly and accurately. The game tracks typing speed and accuracy to provide an interactive, fun experience for users.
